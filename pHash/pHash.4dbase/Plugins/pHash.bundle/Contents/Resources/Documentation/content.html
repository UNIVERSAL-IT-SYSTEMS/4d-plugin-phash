pHash

Plugin Identifier: 20000

Summary

This plugin is a 4D implementation of the pHash open source perceptual hash library.

It can compute and compare the perceptual hashes of PNG, BMP and JPEG files.

You can use it, for example, to search similar images of different file types stored in your database.

Note: Audio and Video files are not supported.

Note: If you create the PNG on Mac OS, make sure the "simple file" option is NOT used. Otherwise the hash value will be zero.

Compute

PH Compute DCT

status:=PH Compute DCT (path; hash)

path (TEXT)

hash (TEXT)

String representation of unsigned 64 bit integer hash.

status (LONGINT)

Less than 0 for error.
